PROJEKT: SystemHelper - Asystent AI do komend systemowych
AUTOR: Jakub Dziemieńczuk
---------------------------------------------------------

OPIS:
Aplikacja umożliwia komunikacje z lokalnym modelem Ollama w celu generowania 
i automatycznego wykonywania komend Windows CMD na podstawie języka naturalnego.

INSTRUKCJA URUCHOMIENIA:
1. Zainstaluj i uruchom aplikacje Ollama (ollama.com).
2. Pobierz model poleceniem w terminalu: ollama run llama3.
3. Otwórz projekt w Qt Creatorze i skompiluj go (Build).
4. Uruchom aplikacje SystemHelper.exe.

URUCHOMIENIE TESTOW JEDNOSTKOWYCH:
1. W Qt Creatorze przejdź do zakładki: Tools -> Tests -> Run All Tests.
2. Wyniki testów (PASS/FAIL) pojawia się w panelu "Test Results".
